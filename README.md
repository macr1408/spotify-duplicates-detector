# Spotify playlist duplicates detector

A tool for detecting duplicated songs within your playlists.

![showcase.gif](showcase.gif)

## Stack Used

- Vite
- React
- Typescript
- Axios
- string-similarity-js
- Tailwind + headlessui
- Docker
